Operator precedence

1 func() function call
2 ! factorial
3 ^ exponentiation
4 + - unary sign operator
6 multiplication without sign note: this means 2km / 3h has the intuitive meaning
7 * · / ÷ // multiplication
8 + - addition

Input number systems

To input hexadecimal, octal, or binary numbers, simply use the prefixes as shown below. Real numbers are supported, using the decimal point.

Roman numerals are also supported by using the prefix 0r. The parser is somewhat relaxed, accepting IIII, VIIII, etc. There are no real numbers, only integers from 1 to 4999.
